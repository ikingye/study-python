    Python 版本管理 #

    pyenv #

    pyenv 加速 #

    在 .zshrc 添加

    function pyenvinstall {
    wget -c https://npm.taobao.org/mirrors/python/$@/Python-$@.tar.xz -P ~/.pyenv/cache/
    pyenv install $@
}

    执行 zsh 使配置生效， 之后即可这样安装： pyenvinstall 3.8.2

    pyenv 基础命令 # 

    # List all available versions
pyenv install -l

# 更新
pyenv update

    pyenv 优先级 #

    • pyenv shell
      • $PYENV_VERSION
    • pyenv local
      • .python-version file
    • pyenv global
      • ~/.pyenv/version
