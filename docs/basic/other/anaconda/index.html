    Anaconda #

    Anaconda 是一个免费开源 的  Python 和  R 语言的发行版本，用于计算科学（数据科学、机器学习、大数据处理和预测分析），Anaconda 致力于简化包管理和部署。

    Anaconda 的包使用软件包管理系统 Conda 进行管理。
